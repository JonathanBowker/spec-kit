# Advanced Analytica Governance Preset

This bundled preset appends Advanced Analytica governance rules to the generated `constitution.md` without modifying Spec Kit core templates.

## What It Adds

- A repository naming protocol for `client-product-function`
- A standard GitHub labelling taxonomy
- Notes marking the current vocabulary and taxonomy as scaffolds that still need final AA values

## Install

```bash
specify preset add aa-governance
```

Use it alongside the bundled `aa-governance` extension when you also want the naming and labelling commands.
